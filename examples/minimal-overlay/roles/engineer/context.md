# Engineer — Role-Specific Context (Example)

Read `product-context.yaml` in the parent directory for full company context.

## Tech Stack

Build backend in Go (Chi router, PostgreSQL).
Build frontend in React + TypeScript + Tailwind CSS.
Use WebSocket for real-time sync (gorilla/websocket).
Use Stripe for payments.

## Codebase Paths

Main project: `product-context.yaml > codebase_paths > project_root`
Architecture decisions: `product-context.yaml > codebase_paths > adrs`

## Handoff Pattern

When your implementation is complete and ready for review, block your kanban task with a structured handoff comment including changed files, test results, and decisions made. The Tech Lead will review and either approve or request changes.
