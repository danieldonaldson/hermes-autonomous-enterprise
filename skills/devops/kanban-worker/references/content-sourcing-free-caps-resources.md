# Content Sourcing — Free CAPS Resources

Reference for seeding a marketplace with free (R0) CAPS-aligned SA teaching resources from open-licensed and public-domain sources.

## License Quick Reference

| Source | License | Can Republish? | Attribution Required |
|--------|---------|----------------|----------------------|
| Siyavula textbooks | CC BY 3.0 | ✅ Yes | "Source: Siyavula (CC BY 3.0). View original at siyavula.com/read/..." |
| DBE past exam papers | Public sector | ✅ Yes | "Source: South African Department of Basic Education" |
| WCED ePortal | Government | ✅ Yes | "Source: WCED ePortal" |
| DBE Rainbow Workbooks | Government | ✅ Yes (Gr R-9 only) | "Source: DBE Rainbow Workbooks" |
| e-Classroom | Proprietary | ❌ Need permission | Contact for cross-listing agreement |
| SchoolClub SA (Pick n Pay) | Proprietary | ❌ Need permission | Contact PnP for permission |
| Thutong (education.gov.za) | Government | ❌ Site defunct — redirected | Previously hosted CAPS curriculum docs |

## Bulk Estimate

- **Siyavula**: 12 textbooks → ~600-1,000 chapter-level listings
- **DBE past papers**: 200-500 exam papers (Gr 12, all subjects)
- **WCED ePortal**: Thousands of resources (lesson plans, worksheets, assessments)
- **DBE Rainbow Workbooks**: Thousands of worksheets (Gr R-9 only — NOT FET)

## Attribution Format

Every listing description includes source attribution. Standard form:

```
Source: [Name] ([License]). View original at [URL].
```

## Cost Model

Free (R0) listings cost effectively R0 to serve. WhatsApp Business API allows free messages within the 24-hour customer session window. The only costs are:
- Storage (negligible — PDFs on disk)
- One-time build effort for the import pipeline
