# Naming Agent Profiles

Agents work better when they have distinct identities. This reference captures
the naming style used for the Clanker Team.

## Style Guide

- **Real person names** (first names, surnames, plausible nicknames)
- **Diverse origins** — draw from different cultures
- **Robot-adjacent** — AI pioneers, fictional robots, mechanical terms
- **A bit funny** — irony and in-jokes work
- **Lowercase** — avoid all-caps codenames
